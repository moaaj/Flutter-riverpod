# Flutter Riverpod Tutorial

A Flutter project built while learning Riverpod state management.

## 🚀 Features

- Riverpod state management
- Product listing UI
- Cart functionality
- Clean Flutter architecture
- Responsive UI
- Beginner-friendly structure

## 📱 Tech Stack

- Flutter
- Dart
- Flutter Riverpod

## 📂 Project Structure

```bash
lib/
 ├── models/
 ├── providers/
 ├── screens/
 ├── widgets/
 └── main.dart
```

## 🛠️ Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio or VS Code
- Emulator or physical device

### Run Locally

```bash
git clone https://github.com/moaaj/Flutter-riverpod.git
cd Flutter-riverpod
flutter pub get
flutter run
```
